# Calculator Dashboard

A comprehensive PHP web application featuring three useful calculators.

## Features

1. **Age Calculator**
   - Calculates exact age in years, months, and days
   - Shows total days and months
   - Calculates days until next birthday

2. **Grade Calculator**
   - Calculates average grade for 5 subjects
   - Shows individual subject remarks
   - Converts to GPA (4.0 scale)
   - Provides final evaluation

3. **BMI Calculator**
   - Calculates Body Mass Index
   - Supports both metric and imperial units
   - Provides BMI category
   - Gives health advice

## Technologies Used
- PHP
- HTML5
- CSS3
- Font Awesome Icons
